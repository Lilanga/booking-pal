Google Service Account Setup

  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable the Google Calendar API
  4. Go to Credentials page
  5. Create a new Service Account
  6. Download the service account key in JSON format
  7. Copy the JSON content and paste it below
  8. Share your calendar with the service account email address
The JSON file should contain your Google Cloud service account credentials including private_key, client_email, and project_id.
